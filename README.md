# ORPALIS PaperScan Professional

## Overview

ORPALIS PaperScan Professional is a document scanning application designed for Windows users. It facilitates the digitization and management of paper documents through advanced scanning capabilities and image processing. The software aims to simplify the conversion of physical documents into organized, searchable digital files.

## Key Features

**Support for various scanner types including TWAIN and WIA**  
**Multi-page document scanning and editing**  
**Image enhancement tools such as deskew, crop, and color adjustment**  
**Optical Character Recognition (OCR) for text extraction**  
**Document annotation and markup capabilities**  
**Export options to common formats including PDF, TIFF, and JPEG**  
**Batch scanning and processing for efficient workflow**  
**Integration with network scanners for shared environments**
